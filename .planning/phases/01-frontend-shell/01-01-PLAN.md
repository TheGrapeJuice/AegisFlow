  <action>
    Initialize the project in the current working directory (do NOT create a subdirectory):

    1. Run `npm create vite@latest . -- --template react-ts` (the dot targets current directory; accept overwrite prompt).
    2. Run `npm install` to install base Vite + React + TypeScript deps.
    3. Run `npm install -D tailwindcss@3 postcss autoprefixer` — use Tailwind v3, NOT v4 (v4 has a different config API and limited ecosystem support as of early 2026).
    4. Run `npx tailwindcss init -p` to generate `tailwind.config.ts` and `postcss.config.js`.
    5. Update `tailwind.config.ts`:
       - `content: ["./index.html", "./src/**/*.{ts,tsx}"]`
       - Enable dark mode: `darkMode: "class"`
       - Extend theme with project colors:
         ```ts
         colors: {
           grid: {
             bg: "#0f1117",
             surface: "#161b27",
             border: "#1e2840",
             text: "#e2e8f0",
             muted: "#64748b",
           },
           node: {
             normal: "#22c55e",
             warning: "#eab308",
             critical: "#ef4444",
           }
         }
         ```
    6. Replace `src/index.css` content with:
       ```css
       @tailwind base;
       @tailwind components;
       @tailwind utilities;

       body {
         background-color: #0f1117;
         color: #e2e8f0;
         font-family: 'Inter', system-ui, sans-serif;
       }
       ```
    7. Update `index.html`: add `class="dark"` to `<html>` tag and set `<title>AegisFlow</title>`.
    8. Remove Vite's default boilerplate from `src/App.tsx` — replace with a simple `<div>Loading...</div>` placeholder (Task 2 will build the real layout).
    9. Remove `src/App.css` — all styling goes through Tailwind.
    10. Install Inter font: add to `index.html` `<head>`:
        ```html
        <link rel="preconnect" href="https://fonts.googleapis.com" />
        <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet" />
        ```
  </action>
