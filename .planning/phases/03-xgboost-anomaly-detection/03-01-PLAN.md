  <action>
Create backend/ml/__init__.py as an empty file (just a package marker).

Create backend/ml/train.py:
- Generates a synthetic dataset of N=5000 samples using the same NODE_STATES sensor ranges
- Features per sample (6 total): voltage, frequency, load, voltage_rolling_mean, voltage_rolling_std, frequency_rolling_mean, frequency_rolling_std, load_rolling_mean, load_rolling_std
  - Wait: that is 9 features. Use exactly these 9 feature names (order matters for inference):
    ["voltage", "frequency", "load", "voltage_mean", "voltage_std", "frequency_mean", "frequency_std", "load_mean", "load_std"]
- Label generation (rule-based, matching simulate_sensor_tick thresholds for reproducibility):
  - is_anomalous = 1 if load > 96.0 or frequency < 59.0 or frequency > 61.0 else 0
  - "warning" range (load 90-96, freq 59.0-59.5 or 60.5-61.0) is labelled 0 (not anomalous) — XGBoost detects critical deviations, not warnings
  - Adjust to produce ~15% positive class: label = 1 if load > 94.0 or abs(frequency - 60.0) > 0.8 else 0 (wider threshold to give model meaningful signal)
- For rolling stats: simulate rolling=5 window by generating voltage/freq/load sequences with slight autocorrelation; compute mean/std from those windows
- Training: xgb.XGBClassifier(n_estimators=100, max_depth=4, use_label_encoder=False, eval_metric='logloss', random_state=42)
- Save model: clf.save_model("backend/ml/model.json")
- Print class balance and a quick accuracy metric after training to verify training succeeded

Create backend/ml/inference.py:
- Module-level: load the model once at import time — _MODEL = xgb.XGBClassifier(); _MODEL.load_model(Path(__file__).parent / "model.json")
- Maintain a per-node rolling buffer: _ROLLING: dict[str, deque] — deque of last 5 (voltage, frequency, load) tuples per node, maxlen=5
- Expose function: predict_anomalies(node_states: dict) -> dict[str, tuple[float, bool]]
  - For each node_id, node in node_states.items():
    - Append (node.voltage, node.frequency, node.load) to _ROLLING[node_id]
    - If len(_ROLLING[node_id]) < 5: emit score=0.0, is_anomalous=False (warm-up period)
    - Else: compute rolling_mean/std from deque contents per field, build feature row, call _MODEL.predict_proba()
    - score = float of positive class probability; is_anomalous = score > 0.5
  - Return dict mapping node_id to (score, is_anomalous)
